Canvas-Paint
=====================
Paint Plug-in for HTML5 using jQuery

jQuery Version 2.1.0

Browsers Currently Tested
-------------------------
Firefox 27

Current Functionality
-------------------------
 * Mouse Movement Detection on Canvas 
 * Pencil Tool
 * Circle Tool
